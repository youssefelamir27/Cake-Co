A modern landing page website for **Cake & Co**, a fictional bakery and dessert shop specializing in cakes, cookies, and authentic Middle Eastern sweets such as Kahk, Petit Fours, Qatayef, and Halva.

**Live Demo**: (https://youssefelamir27.github.io/Cake-Co/)

## Overview

Cake & Co is a static website designed to showcase a premium bakery brand. Key highlights include:

- Fresh, high-quality desserts with fast delivery
- Customization options for birthdays, weddings, and special events
- Inclusive menu (vegan, gluten-free, keto options)
- Featured products: Pistachio Cookies, Honey Cake, Semolina Halva, Cheese Qatayef, Plain Kahk, and Petit Fours boxes
- Store locations across Cairo, Alexandria, and Hurghada
- Brand stats (11 stores, 22 awards, etc.) and team introduction

This project serves as a front-end portfolio piece demonstrating clean HTML structure and CSS styling.
